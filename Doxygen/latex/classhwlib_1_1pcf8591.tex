\subsection{Detailed Description}
\hyperlink{classhwlib_1_1pcf8591}{pcf8591} I2C A/D and D/A converter

This class implements an interface to a \hyperlink{classhwlib_1_1pcf8591}{pcf8591} A/D \& D/A converter chip.



A \hyperlink{classhwlib_1_1pcf8591}{pcf8591} is an I2C slave that provides 4 8-\/bits A/D inputs (that can be arranged in various combinations of single and balaced inputs), and a single8-\/bits D/A output. The power supply range is 2.\+5 .. 5.\+5 Volt. Of the 7-\/bit slave address, 3 bits are set by the level of 3 input pins (A0..A2) of the chip. When all 3 are pulled low, the slave address is 0x48.

The next code repeatedly prints the values read by the 4 (single-\/ended) A/D converters.
