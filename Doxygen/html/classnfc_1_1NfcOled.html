Detailed Description

Oled decorator for the NFC class.

With a decorator, additional features can be added on top of the already existing NFC funcions. This specific decorator does that by executing a command, and displaying ( most of ) it's output on an oled display. Not all functions make use of the oled display (yet). Those functions will just call their slave counterfit functions
