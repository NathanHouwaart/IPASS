Abstract function to decrement a valueblock by a given value ( value )

Note
When this function is implemented, the actual valueblock of the card will not be modified. Only the internal temporary buffer will be updated. In order to update the actual valueblock the transfer function needs to be called after this function. source: http://www.cs.ru.nl/~wouter/papers/2008-thebest-updated.pdf p. 7 2.2 using value blocks
Parameters
cardinfoA card class where the card data can be stored in
cardNumberCard that needs to be written to
AorBAutenticate with key a or key b
pageNumberPagenumber that needs to be authenticated
sectorSector where the new valueblock will be located in
keyPointer to key array that the sector trailer block needs to be autenticated with
valueValue the sectorblock needs to be decremented by
Returns
statusCode Status of the operation

Implemented in nfc::PN532_chip, and nfc::NfcOled.
