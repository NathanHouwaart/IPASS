Detailed Description

This file implements a ringbuffer, abstract UART interface and an implementation of this interface.

Note
The hardware UART implementation is only working when sending a command to a slave. Reveiving data over the UART bus is not yet possible. In order to make this happen, cpu interrupts need to be added tho interrupt the cpu when RXRDY register is 1. This is however out of my knowledge and therefore not yet implemented. Also this file is not yet split up in to .cpp files and .h files. This will be done as soon as the receive function is working
