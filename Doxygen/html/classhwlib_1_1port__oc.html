virtual void hwlib::port_oc::flush ( )
pure virtual

flush the port buffer

This function flushes the port write buffer: if a buffered value has not yet been written it is written to the pin. When the value has been written, it might be written again.

The pin write buffer might also be flushed as a side-effect of other operations.

Implemented in hwlib::port_oc_from_pins_t, hwlib::port_oc_from_oc_t, hwlib::port_invert_from_oc_t, hwlib::port_direct_from_oc_t, and hwlib::pcf8574.

virtual uint_fast16_t hwlib::port_oc::read ( )
pure virtual

read from the port

This function reads and returns the pins that are part of the port. The lowest bit of the result reflects the first pin of the port, etc.

Before calling this function the port direction must have been set to input by calling port_direction_set_input().

Implemented in hwlib::port_oc_from_pins_t, hwlib::port_oc_from_oc_t, hwlib::port_invert_from_oc_t, hwlib::port_direct_from_oc_t, and hwlib::pcf8574.

refresh the port buffer

This function refreshed the contents of the read buffer that might be used by get_buffered() with a value read from the pins.
