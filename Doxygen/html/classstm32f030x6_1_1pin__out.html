stm32f030x6 pin_out constructor

Constructor for an stm32f030x6 output pin.

This constructor sets the pin direction to output.

This constructor doesn't set the pin value to high or low, the set function must be called to do so.

write the pin

This function sets the level of the pin to the value v. A value of true makes the pin high, a value of false makes it low.

The actual effect on the pin might be postponed until the next flush() call.
